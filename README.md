# Personal Neovim Config 
this is basic config  made of 11 plugins  to learn neovim which includes,
1- lspconfig +mason 
2- mini nvim basic configs 
3- nvim treesitter (for highlights)
4-rosepine theme 
5- markdown file renderer 
## requirements
1- "C" Compiler -> gcc ,or zig 
2- nerdfonts  to show icons 
3- packages needed for telescope fzf,ripgrep,fd ,Curl
